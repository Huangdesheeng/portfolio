作品集部署说明
===============

方法一：GitHub Pages（推荐，免费永久）
------------------------------------

1. 注册/登录 GitHub 账户
   访问：https://github.com

2. 创建新仓库
   - 点击右上角 "+" 号
   - 选择 "New repository"
   - 仓库名称：portfolio（或你喜欢的名字）
   - 选择 "Public"
   - 勾选 "Add a README file"
   - 点击 "Create repository"

3. 上传文件
   - 在仓库页面点击 "uploading an existing file"
   - 拖拽以下文件到页面：
     * index.html
     * 头像2.jpg
     * 二维码.jpg
   - 点击 "Commit changes"

4. 启用 GitHub Pages
   - 进入仓库的 "Settings" 选项卡
   - 左侧菜单找到 "Pages"
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，文件夹选择 "/ (root)"
   - 点击 "Save"

5. 获取链接
   - 等待1-2分钟
   - 刷新 Pages 设置页面
   - 你会看到类似这样的链接：
     https://你的用户名.github.io/portfolio/

------------------------------------

方法二：Netlify（简单快速）
------------------------------------

1. 访问 https://app.netlify.com
2. 注册/登录账户
3. 直接拖拽整个 "新作品集" 文件夹到页面
4. 自动部署后会生成一个链接
5. 可以在设置中自定义子域名

------------------------------------

方法三：Vercel
------------------------------------

1. 访问 https://vercel.com
2. 注册/登录账户
3. 导入 Git 仓库（需要先推送到 GitHub）
4. 自动部署后生成链接

------------------------------------

当前文件：
- index.html（主页面）
- 头像2.jpg（简介头像）
- 二维码.jpg（微信二维码）

本地预览：
- 直接双击 index.html 打开
- 或运行 start-server.bat 后访问 http://localhost:8000
