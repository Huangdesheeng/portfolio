# 黄德生 - 视觉设计师作品集

在线预览：https://你的用户名.github.io/portfolio/

## 本地预览

直接双击 `index.html` 文件即可在浏览器中查看。

## 部署到 GitHub Pages

1. 在 GitHub 创建新仓库（名称：portfolio）
2. 上传所有文件
3. 进入 Settings -> Pages -> 启用
4. 等待1-2分钟即可访问

## 文件说明

- `index.html` - 主页面
- `头像2.jpg` - 个人头像
- `二维码.jpg` - 微信二维码
- `start-server.bat` - 本地服务器启动脚本
- `部署说明.txt` - 详细部署指南
- `快速部署.bat` - 快速部署助手
