# 作品集部署指南

## 最快方式：GitHub Pages

### 步骤1：创建 GitHub 账户
访问 https://github.com 注册账户

### 步骤2：创建新仓库
1. 点击右上角 "+" -> "New repository"
2. 仓库名：`portfolio`
3. 选择 "Public"
4. 点击 "Create repository"

### 步骤3：上传文件
在仓库页面：
1. 点击 "uploading an existing file"
2. 拖拽以下文件：
   - `index.html`
   - `头像2.jpg`
   - `二维码.jpg`
3. 点击 "Commit changes"

### 步骤4：启用 GitHub Pages
1. 进入仓库 "Settings"
2. 左侧菜单找到 "Pages"
3. Source 选 "Deploy from a branch"
4. Branch 选 "main"，文件夹选 "/ (root)"
5. 点击 "Save"

### 步骤5：获取链接
等待1-2分钟，刷新页面，你会看到：
`https://你的用户名.github.io/portfolio/`

## 备选方案

### Netlify（更简单）
1. 访问 https://app.netlify.com
2. 直接拖拽整个文件夹
3. 自动生成链接

### 本地预览
直接双击 `index.html` 文件
